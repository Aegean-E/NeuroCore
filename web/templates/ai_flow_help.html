Quick Start

1. Drag functions from the sidebar onto the canvas

2. Connect nodes by dragging from output dots to input dots

3. Click a node to configure it

4. Save your flow and run it from the Chat

Node Types

Input Nodes

Chat Input, Telegram Input - Start of conversation flow

Processing Nodes

LLM Core, System Prompt, Memory, Knowledge - Transform and process data

Logic Nodes

Delay, Repeater, Conditional Router - Control flow execution

Output Nodes

Chat Output, Telegram Output, Tool Dispatcher - Send results

Keyboard Shortcuts

Delete Remove selected
Ctrl+A Select all
Ctrl+Z Undo
Space+Drag Pan canvas
Scroll Zoom in/out
Double-click Edit annotation

Goal System

The Goals feature enables autonomous agents:

  • Use SetGoal tool to create a goal
  • Use Check Goal node to inject goal into context
  • Use MarkGoalComplete tool when done
  • Use DeleteGoal tool to remove a goal

Example flow: Chat Input → Check Goal → LLM Core → Tool Dispatcher → Chat Output

Memory System

Memory Recall - Searches vector database for relevant context

Memory Save - Stores interactions to long-term memory

Configure thresholds in Settings → Modules → Memory

Tools

Enable tools in System Prompt node config:

1. Click System Prompt node → 2. Select tools → 3. Save

Custom tools can be created in Settings → Tools

Tool Dispatcher

Executes tools enabled in System Prompt:

Flow order:

System Prompt → LLM Core → Tool Dispatcher → (Tool executes) → LLM Core → ...

LLM decides which tool to call, Tool Dispatcher executes it, result returns to LLM.

Available: SetGoal, MarkGoalComplete, Weather, Calculator, etc.

Multiple Active Flows

Run multiple flows simultaneously:

  1. Set a flow as active in Settings → AI Flow
  2. Use Repeater node for scheduled tasks
  3. Each flow runs independently
